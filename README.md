## Installation

```bash
$ brew install pkg-config cairo pango libpng jpeg giflib librsvg pixman
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
